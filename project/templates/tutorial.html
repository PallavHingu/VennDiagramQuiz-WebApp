Tutorial


How to Use the Application

Login

Once greeted with the main page, you must use your login details to login to the application. If however you don’t have an account, sign up.


For Teachers

Once signed in, the home screen will pop up. On this screen, you will be able to access the leaderboard to track how students are doing, or add more questions to the easy or hard category. There is also a ‘manage all question’ button to see all the questions, their difficulties and their answers. In case of the question being incorrect, it can be edited or even deleted in this tab.

To add new questions, click on the ‘Add Easy Qs’ or ‘Add Hard Qs’ buttons. Here there will be a text input box, where you can enter the question to be added, and a model of a venn diagram - click on the sections of the venn diagram to highlight them. Once the venn diagram has been shaded in properly to match the question input, click the ‘Add Question’ button to insert this new question into the list of questions.


For Students

Once signed in, the home screen will pop up. On this screen, you will be able to access the leaderboard to track how you and other students are doing, or choose what type of question you would like to work on. Simply choose ‘Easy’ or ‘Hard’ to start doing questions of that difficulty type. Click on the sections of the venn diagram to highlight them according to the question. Once the questions have been completed, click Home in the top right corner to exit back to the home screen.



Sets Information

Sets

A set is an unordered collection of different elements. Curly brackets are used to represent sets. Examples of sets contain the set of natural numbers, or the set of odd numbers etc.


Special Sets

U - Universal set (the set of all objects under consideration) Sdlkfj - Empty set (the set of no objects)

A subset:
If every element of X is an element of set Y, set X is a subset of subset Y (it is written as X⊆Y)

For example, if we had set X = {1,2} and set Y = {1,2}, then set X is a subset of Y. Another example would be if we had set X = {1,2} and set Y = {1,2,3,5}. In this case set X is a proper subset of Y.
A proper subset is defined as a set that is a subset of another, but not equal to the set. Here, we see that set X and set Y aren’t the same, but set X is a subset - this means it qualifies to be called a ‘proper subset’.


Set Operation

Set Operations include Set Union, Set Intersection, Set Difference, Complement of Set, and Cartesian Product.


Set Union

The union of sets X and Y is the set of all elements in X and Y (written as X U Y).
For example, if we let set X = {1,2,3} and set Y = {3,5,6,7,8}, then X U Y = {1,2,3,5,6,7,8}. Notice that set X and set Y both have 3, but X U Y doesn’t have 2 occurrences of 3s. This is because there are no repeated elements in a set.


Set Intersection

The intersection of sets X and Y is the set of elements that occur in both X and Y (written as X ∩ Y).
For example, if we let set X = {1,2,3,5} and set Y = {3,5,6,7,8}, then X ∩ Y = {3,5}.


Complement of a Set

The complement of a set (shown as X’), is the set of elements which aren’t a part of set X. It can be written as X’ = { x | x ∉ Y }
For example, if we let X = the set of all odd numbers, X’ would be the set of all numbers which aren’t odd


Set Difference

Denoted by X - Y, it is the set of elements which are only in X but not in Y. This can also be written as
X - Y = { x | x ∈ X AND x ∉ Y }.

For example, if X = {1,2,3,4,5} and Y = {4,5,6,7,8}, X - Y = {1,2,3}.
Y - X = {6,7,8} - - > This shows that (X - Y) ≠ (Y - X)


Cross Product

The Cartesian product of two sets X and Y, denoted by X × Y, is the set of all ordered pairs (x,y), such that x is an element of X and y, an element of Y.


Venn Diagram

Venn Diagrams are schematic diagrams to show the relationship between sets. We can thus use these to represent the most of the operations from above in a way that makes sense more visually.
